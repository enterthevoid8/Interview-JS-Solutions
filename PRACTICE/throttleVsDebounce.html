⚡ Simple Debounce & Throttle

1. Debounce - "Wait until they're done"

// DEBOUNCE: Waits for pause before executing function debounce(func, delay) { let timer; return function(...args) { clearTimeout(timer); timer = setTimeout(() => func(...args), delay); }; }
📌 Key Idea: Resets the timer every time the function is called. Only executes after the specified delay with no new calls.

Demo: Search Box

Type something - API call happens 1 second after you stop typing

API Calls: 0

Without debounce, typing "hello" would make 5 API calls!
With debounce, it only makes 1 call after you finish.

2. Throttle - "Maximum once per X seconds"

// THROTTLE: Limits execution to once per time period function throttle(func, delay) { let lastCall = 0; return function(...args) { const now = Date.now(); if (now - lastCall < delay) return; lastCall = now; func(...args); }; }
📌 Key Idea: Executes immediately, then ignores calls until the delay period passes. Guarantees maximum execution rate.

Demo: Button Click

Click the button as fast as you can - it only fires once per second

Executions: 0

Prevents spam clicking! Perfect for preventing double-submissions.

4. When to Use Each

✅ Use Debounce

  • Search input
  • Auto-save text
  • Form validation
  • Window resize (final)
  • Autocomplete

✅ Use Throttle

  • Scroll events
  • Mouse movement
  • Button clicks
  • Window resize (during)
  • Game updates
🎯 Simple Rule:
Debounce = Wait for user to finish
Throttle = Limit how often something happens
